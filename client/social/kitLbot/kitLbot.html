Kit L-BOT

Permite realizar distintas construcciones a partir de la combinación de piezas encastrables (ladrillos) con componentes electrónicos.



El kit incluye: una interfaz gráfica de programación por bloques que permite darle órdenes de manera sencilla al robot, un kit de expansión con sensores que amplían sus funcionalidades y material didáctico para utilizar en el aula.

T-BOT provee el programa Ardublock para programar el kit utilizando bloques, que además permite ver el código fuente de un programa.
 

Programa

Con Material Didáctico que incluye actividades centradas en el área de matemática a partir de disparadores temáticos, guías de armado de las diferentes construcciones y programas de ejemplo.

Manterial didáctico

¿Qué se puede hacer
con este Kit?

T-BOT permite realizar construcciones mecánicas simples como molinos de viento, puentes y grúas; y robots más complejos centrados en la movilidad y las mediciones a través de sensores. Además, sus piezas encastrables promueven la creatividad de los alumnos al permitirle realizar sus propios diseños.
